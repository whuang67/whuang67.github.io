Website Applications

  • Web Develop: Score Keeper
    • Target Score: The default value is 5. You may also type in a user-defned one.
    • How to Win: The first player who reach the target score first win the game.
    • Reset: Click RESET to reset the game. When you change the target score, game will be reset automatically.
    • Records: Previous game results and total score are recorded. Refresh this page to clear records.

  • Web Develop: To Do List
    • Adding: Type in the next event into the "Add New Todo" area and then hit "Enter".
    • Hiding: Hit "+" sign to hide the "Adding" input section, and re-hit to redisplay it.
    • Removing: Click the "trash can" sign in front of an event to remove it.
    • Erasing: Click the event to erase an event temporarily, and re-click to cancel it.

  • Web Develop: Color Game
    • How to Win: The goal of this game is to find out the color that the RGB code represents.
    • Easy/Hard: There will be 3 and 6 choices in Easy and Hard versions, respectively.
    • Incorrect Selection: The wrong color square will disappear after being clicked if it is not the correct one.
    • Reset: Click "NEW COLORS" to reset the game, and you will get 3 or 6 new colors and 1 new target color.

  • Data Science Consulting: Air Pollution in Shanghai - Analyses and Predictions
    • Login: Both username and password are group5.
    • Visualization: Plots of PM 2.5 vs variables. Lower y limits are encouraged due to outliers.
    • Prediction: Upload a csv file containing the weather information of the next 24 hours to make predictions.
    • Sample File: Click here to download a sample csv file. Files used in the Prediction must follow its format.
    • Calculator: Type in and select values of corresponding variables manually can also provide predictions.

Data Science Projects

  • Consulting: Air Pollution in Shanghai - Analyses and Predictions
    • Imputed missing values of continuous variables by applying median substitution method
    • Performed multivariate analyses including ANOVA, Tukey's HSD test and Chi-square
    • Fit predictive models by Random Forest algorithm after features being selected by variable importance
    • Built a Shiny website application to help client educate the public and make future PM 2.5 predictions

  • Data Mining: KDD Conference Paper Analysis
    • Extracted authors of each paper, converted raw KDD papers into a matrix of TF-IDF features, and reduced dimensionality
    • Applied DBSCAN algorithm to group papers into different clusters and perform outlier detection
    • Performed Apriori algorithm to discover frequent patterns and association rules between different authors

  • Consulting: Tickets Redemption Rate Analysis
    • Imputed missing values of variable "Capacity" by applying regression method
    • Fit logistic regression and decision tree models, summarized the results from different models
    • Built a Shiny website application to help summarize data visualization and variable correlationship

  • Statistical Learning: Allstate Claims Severity
    • Performed feature selection by building LASSO and Random Forest models, and dimensionality reduction using PCA
    • Predicted variable “loss” using parametric (Linear Regression, LASSO) and non-parametric (Random Forest) methods
    • Applied ensemble methods at the final stage to obtain the model with the best performance
