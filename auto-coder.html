AutoCoder
This is a tool that can automatically construct simple variables and assign formats to them created by Wenke Huang, Statistician at the University of California, Los Angeles. SAS, R, Python, Stata and SPSS are currently available.
Click README for more information.
    Collapse (input: categorical, output: categorical)
  • Collapse is for combining several original levels of input variable into new levels
  • The levels that you would like to combine should be seperated by semicolons ";". Click Add another level to add one more level. The levels of input variables that are not included will be missing in the corresponding new variables
  • For descrete input variables (integers, etc), please check the top-right checkbox
  • Format will be provided automatically if missing
  • If duplicate values are detected, you will not be able to add new levels or generate codes
    Categorize (input: continuous, output: categorical)
  • Categorize is for converting continuous variables into categorical ones
  • Please select the lower and upper boundaries of the your categories. Right-open intervals are used here. The min value of only the first level only can be missing, and so can the max value of the last level if there are more than one levels
  • Same as Collapse, format will be provided automatically if missing
  • Input values must be monotonically increasing, or you will not be able to add new levels or generate codes
    Up/Bottom Code (input: continuous, output: continuous)
  • Up/Bottom Code is for relimiting the range of continuous variables
  • Please select the lower and upper range of the variable. The values less than the lower bound will be converted to the lower bound while the values greater than the upper bound will be the upper bound.
  • Unlike Collapse or Categorize, formats will be provided automatically and only for SAS
  • Input values must be monotonically increasing if not missing, and they cannot be missing together
    General
  • You can remove a variable but the last one any time you would like
  • Don't forget to choose the languages that you prefer. Formats are NOT available for R or Python
  • When ready, please click Generate Codes!. Fields will turn gray once you submit successfully
  • Any questions or error, please contact huangwk223ATgmailDOTcom
